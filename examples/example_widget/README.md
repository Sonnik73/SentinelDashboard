# Example Widget Module

This is a minimal SentinelDashboard module example.

To install it:

1. Copy this folder to `modules/example`
2. Restart SentinelDashboard
3. Open `/api/example`
4. Add `example` widget to a view

Required file:

- `manifest.json`

Optional files:

- `service.py`
- `api.py`
- widget template
- static assets
