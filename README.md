🛡️ SentinelDashboard

SentinelDashboard — это информационная панель, работающая на Raspberry Pi и предназначенная для отображения актуальной информации на планшетах или других устройствах в локальной сети.

Проект создаётся как модульная система мониторинга и информационный центр для рабочего помещения.

---

Основные возможности

🖥️ Мониторинг Raspberry Pi

- Загрузка процессора
- Использование оперативной памяти
- Температура процессора
- Время работы системы

🌤️ Погода

Отображение погоды для выбранных городов.

Планируемые города:

- Москва
- Краснодар
- Ульяновск

---

📰 Новости

Поддержка RSS-лент.

Планируется подключение:

- Хабр
- РИА Новости
- SecurityLab
- OpenNET

---

🎂 Дни рождения

Импорт сотрудников из Excel.

Возможности:

- Сегодняшние дни рождения
- Ближайшие события
- Уведомления

---

🌐 Мониторинг сети

- Доступность Интернета
- Ping
- Скорость соединения
- Количество Wi-Fi-клиентов (при поддержке оборудования)

---

📷 Видеонаблюдение

Планируется отображение RTSP-потоков с камер после подключения видеорегистратора.

---

Архитектура проекта

- Backend: Python + FastAPI
- Frontend: HTML + CSS + JavaScript
- Шаблоны: Jinja2
- Хранение данных: SQLite (в будущем)
